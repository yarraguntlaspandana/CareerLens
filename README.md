# CareerLens

CareerLens is a web-based analytics dashboard that helps students and job seekers understand current job market trends through data visualization.

## Features

- Analyze trending skills
- Explore hiring locations
- View experience-level demand
- Interactive analytics dashboard
- Career insights through data visualization
- User-friendly interface

## Tech Stack

- Node.js
- Express.js
- EJS
- Chart.js
- JavaScript
- JSON
